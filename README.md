# Laboratorio de diagnóstico empresarial  
**Alonso & Espinosa · Consultoría Operativa**

Este repositorio contiene el **Laboratorio de diagnóstico empresarial**, una colección de evaluaciones breves diseñadas para ayudar a pequeñas y medianas empresas a detectar posibles problemas operativos en su organización.

Las herramientas están pensadas como una primera exploración orientativa antes de abordar mejoras estructurales en procesos, organización o tecnología.

---

## Objetivo del laboratorio

Muchas pequeñas empresas funcionan razonablemente bien, pero con señales recurrentes como:

- Dependencia excesiva de una persona clave
- Sobrecarga administrativa
- Procesos poco claros o poco definidos
- Herramientas digitales poco integradas
- Falta de visibilidad sobre márgenes o resultados

El laboratorio reúne **evaluaciones breves (2–3 minutos)** que permiten detectar estas señales de forma rápida.

---

## Evaluaciones disponibles

El laboratorio incluye actualmente seis herramientas:

1. **Diagnóstico operativo rápido**  
   Identifica señales de desorden operativo o falta de estructura interna.

2. **Evaluador de madurez digital**  
   Analiza el nivel de organización digital y el uso de herramientas.

3. **Evaluación de carga operativa**  
   Detecta posibles cuellos de botella administrativos.

4. **Evaluación de dependencia del fundador**  
   Mide hasta qué punto la empresa depende de una sola persona.

5. **Evaluación de claridad de procesos**  
   Analiza si los procesos de trabajo están definidos y compartidos.

6. **Evaluación de control del negocio**  
   Explora el nivel de visibilidad sobre resultados y márgenes.
